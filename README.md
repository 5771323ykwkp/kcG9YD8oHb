# Python旅游景点分析系统 python30

本项目是一款基于Python的旅游景点分析系统，集成了Flask框架、sqlite数据库、HTML、CSS以及JavaScript技术。系统旨在为用户提供一个稳定、高效的平台，对旅游景点数据进行多维度分析和展示。

## 技术栈

- 后端：Python Flask
- 数据库：sqlite
- 前端：HTML + CSS + JavaScript

## 功能模块

- 登录注册
- 旅游数据总览
- 旅游景点词云分析
- 旅游景点信息管理
- 旅游景点评论可视化分析

## 系统角色

- 用户：登录、注册、浏览旅游景点信息、查看旅游景点评论
- 管理员：管理旅游景点信息、查看用户评论

## 运行环境

- Python 3.6 或以上版本
- Flask 框架
- sqlite 数据库

## 部署步骤

1. 确保已安装Python环境和相关依赖库
2. 克隆或下载项目源码
3. 安装项目依赖：`pip install -r requirements.txt`
4. 配置数据库和项目相关设置
5. 运行开发服务器：`python manage.py runserver`
